矩阵的敏感性分析

  • error

\[ e=x - x^* = A^{-1}(b - Ax_*) \]

If A is nonsingular

  • residual \(r = b - Ax^*\)

  • 条件数condition number

\[ M = max \frac{||Ax||} {||x||} \\ m = min \frac{||Ax||} {||x||} \]

M=m is called the condition number of A \[ \kappa (A) = \frac {M}{m} \]

  • relative error

  • relative residual

  • cond(A) or cond(A,2) computes$ _2(A)$.. Uses svd(A). Suitable for smaller matrices where the geometric properties of the l2-norm are important.

  • cond(A,1) computes$ _1(A)$.. Uses inv(A). Less work than cond(A,2)

  • cond(A,inf) computes $ _1(A)$. Uses inv(A). Same as cond(A’,1).

  • condest(A) estimates$ _1(A)$, Uses lu(A) and a recent algorithm of Higham and Tisseur [9]. Especially suitable for large, sparse matrices.

  • rcond(A) estimates \(1/ \kappa_1(A)\). Uses lu(A) and an older algorithm developed by the LINPACK and LAPACK projects. Primarily of historical interest
